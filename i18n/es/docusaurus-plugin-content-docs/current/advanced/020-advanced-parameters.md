# ⚙️ Parámetros avanzados

Puedes usar parámetros de cadena de consulta para personalizar cómo se comporta el cliente del juego QuizWitz. Estos parámetros pueden añadirse a cualquier enlace de juego usando la función **Configuración avanzada del juego**.

Ejemplo:

https://play.quizwitz.com/13305:qyHBEVVBqT?theme=emerald

📘 [¿Qué son las cadenas de consulta?](https://en.wikipedia.org/wiki/Query_string)

---

## Parámetros disponibles:

| Parámetro                |           Por defecto          |           Ejemplo           | Explicación                                                                                                                                                                                                                                      |
| ------------------------ | :----------------------------: | :-------------------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `language`               | (navegador) |              en             | Código de idioma ISO-639 para cargar y usar como idioma base                                                                                                                                                                                     |
| `theme`                  |             quizted            |           emerald           | Nombre (o URL aprobado) del tema a cargar                                                                                                                                                                                     |
| `reservation`            |                /               |            abcdef           | Token de reserva a usar (en juegos en vivo)                                                                                                                                                                                   |
| `remote`                 |   quizwitz.tv  | quizwitz.tv | Servidor remoto CatLab a usar                                                                                                                                                                                                                    |
| `server`                 |                /               |              10             | ID del servidor remoto CatLab a usar (con auto descubrimiento)                                                                                                                                                                |
| `publisher`              |                /               |           QuizWitz          | Nombre del editor que está alojando el juego. Esto se usa para personalizar vistas                                                                                                                                               |
| `smileys`                |                1               |              0              | Configura en 0 para desactivar los emoticonos en el juego                                                                                                                                                                                        |
| `outroPlayers`           |               12               |          5,4,3,1,2          | Define la cantidad (número) O el orden de jugadores (lista separada por comas de posiciones) que serán anunciados durante la despedida del juego.                                          |
| `focusPositions`         |                /               |            50,100           | Define una lista de posiciones adicionales que se mostrarán en la aplicación quizmaster                                                                                                                                                          |
| `translations`           |                1               |              0              | Configura en 0 para desactivar la carga de traducciones del cuestionario que estás cargando                                                                                                                                                      |
| `cycleTranslations`      |                0               |              1              | Configura en 1 para alternar entre todos los idiomas disponibles del cuestionario por pregunta                                                                                                                                                   |
| `showLongQuestions`      |                0               |              1              | Configura en 1 para mostrar la 'pregunta larga' en la pantalla del juego                                                                                                                                                                         |
| `forcePiecharts`         |                0               |              1              | Configura en 1 para mostrar siempre toda la retroalimentación en gráficos circulares                                                                                                                                                             |
| `forceNoPiecharts`       |                0               |              1              | Configura en 1 para nunca agrupar toda la retroalimentación en gráficos circulares.                                                                                                                                              |
| `piechartPercentages`    |                0               |              1              | Configura en 1 para mostrar porcentajes en lugar de valores absolutos en todos los gráficos circulares                                                                                                                                           |
| `monitors`               |                /               |            nl,fr            | Si está configurado, en juegos en vivo se crearán códigos separados para mostrar un 'monitor' en ese idioma específico para presentadores localizados.                                                                           |
| `allowLogin`             |                1               |              0              | Configura en 0 para impedir que los usuarios inicien sesión                                                                                                                                                                                      |
| `tracker`                |                1               |              0              | Configura en 0 para desactivar todo el seguimiento. No habrá reporte del quiz disponible                                                                                                                                         |
| `random`                 |                0               |              1              | Configura en 1 para cargar un 'quiz aleatorio'                                                                                                                                                                                                   |
| `delay`                  |                0               |            30000            | Establece la cantidad de milisegundos que se retrasará toda interacción del jugador (para transmisiones en vivo)                                                                                                              |
| `countdown`              |               10               |              60             | Establece la cantidad de segundos que el juego hará 'cuenta regresiva' en modo presentación.                                                                                                                                     |
| `autoCountdown`          |                0               |              1              | Configura en 1 para iniciar automáticamente la cuenta regresiva después de que se unan los primeros jugadores en modo presentación.                                                                                              |
| `autoRestart`            |                0               |              1              | Configura en 1 para reiniciar automáticamente el juego después de que termine.                                                                                                                                                   |
| `waitForPlayers`         |                0               |              1              | Configura en 1 para no esperar a ningún jugador cuando `autoCountdown` está activado                                                                                                                                                             |
| `askEmail`               |                1               |              0              | Configura en 0 para no solicitar la dirección de correo electrónico del usuario en modo presentación.                                                                                                                            |
| `beacon`                 |                /               |           my-beacn          | Configura un token de baliza CatLab Remote que puede usarse para conectar automáticamente la aplicación quizmaster.                                                                                                              |
| `rounds`                 |                5               |              7              | Configura la cantidad de rondas que se generarán en un quiz aleatorio.                                                                                                                                                           |
| `questions`              |                7               |              7              | Configura la cantidad de preguntas que se generarán para cada ronda en un quiz aleatorio.                                                                                                                                        |
| `showListenQuotes`       |                1               |              0              | Configura en 0 para desactivar las citas "por favor escucha" divertidas.                                                                                                                                                         |
| `shared`                 |                /               |  123:abcdef | Token de acceso de una entrada compartida.                                                                                                                                                                                       |
| `music`                  |                1               |              0              | Configura en 0 para desactivar toda la música (del juego). El audio subido seguirá reproduciéndose.                                                                                           |
| `connectMusic`           |                1               |              0              | Configura en 0 para desactivar la música (del juego) que se reproduce durante la fase de 'conexión'.                                                                                                          |
| `slideshowVideoInterval` |               300              |             300             | Cuando se suben vídeos en la fase de pantalla de conexión, esto define la cantidad de segundos entre cada reproducción de vídeo.                                                                                                 |
| `slideshowImageInterval` |               20               |              60             | Cuando se suben imágenes en la fase de pantalla de conexión, esto define la cantidad de segundos que cada imagen será mostrada.                                                                                                  |
| `skipOnAllAnswered`      |                1               |              0              | Configura en 0 para sobrescribir el valor de `skipOnAllAnswered` de las entradas                                                                                                                                                                 |
| `departments`            |                1               |            A,B,C            | Configura en 0 para desactivar la carga de departamentos. Configura con una lista de nombres separados por comas para asignar automáticamente a todos los jugadores que se conecten a un departamento aleatorio. |
| `showRankInDepartment`   |                1               |              0              | Configura en 0 para impedir que los usuarios vean su rango dentro de su departamento.                                                                                                                                            |
| `showDepartmentRanking`  |                1               |              0              | Configura en 0 para desactivar la visualización del ranking por departamentos entre rondas.                                                                                                                                      |
| `preloadVideo`           |                0               |              1              | Configura en 1 para forzar la precarga de todos los fragmentos de vídeo.                                                                                                                                                         |
| `n`                      |                /               |          `_prompt_`         | Establece (o solicita configurando en `_prompt_`) un nombre para el grupo de jugadores que está jugando el juego. Este nombre se envía al reporte del quiz.                                   |

---

## 💡 Consejos de uso

- Se pueden combinar múltiples parámetros usando `&`
- Usa estas opciones con **Configuración avanzada del juego** al compartir o incrustar enlaces
- Muchas opciones son útiles para optimización de transmisiones en vivo o eventos multilingües
